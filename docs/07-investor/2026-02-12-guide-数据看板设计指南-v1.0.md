## 一、看板定位

### 1.1 目标用户

| 用户类型 | 使用场景 |
|---------|---------|
| 创始人/CEO | 战略决策，融资展示 |
| 运营团队 | 日常监控，优化策略 |
| 投资人 | 了解业务健康度 |

### 1.2 核心价值

- **实时监控**：掌握业务动态，快速发现问题
- **数据驱动**：用数据说话，指导决策
- **展示证明**：向投资人证明运营能力

## 二、指标体系

### 2.1 核心指标（北极星指标）

| 指标 | 定义 | 目标值 | 优先级 |
|-----|------|-------|-------|
| **累计领养数** | 平台成立至今完成的领养总数 | - | ⭐⭐⭐⭐⭐ |
| **本月新增** | 当月新增领养数 | 环比增长 20% | ⭐⭐⭐⭐⭐ |
| **领养转化率** | 申请→完成领养的比例 | >15% | ⭐⭐⭐⭐⭐ |
| **平均匹配时长** | 用户注册到首次匹配的天数 | <3天 | ⭐⭐⭐⭐ |
| **用户满意度** | 领养用户评分（5分制） | >4.5 | ⭐⭐⭐⭐ |

### 2.2 用户指标

| 指标 | 定义 | 目标值 |
|-----|------|-------|
| 日活（DAU） | 日活跃用户数 | - |
| 月活（MAU） | 月活跃用户数 | - |
| 次日留存率 | 次日回访用户比例 | >40% |
| 7日留存率 | 7日内回访用户比例 | >20% |
| 新用户数 | 当日/周/月新增注册用户 | - |

### 2.3 业务指标

| 指标 | 定义 | 目标值 |
|-----|------|-------|
| 发布宠物数 | 送养人发布的宠物数量 | - |
| 咨询量 | 领养咨询消息数 | - |
| 申请量 | 领养申请提交数 | - |
| 完成量 | 成功完成领养的订单数 | - |
| 拒绝率 | 申请被拒绝的比例 | <20% |

### 2.4 财务指标

| 指标 | 定义 | 目标值 |
|-----|------|-------|
| GMV | 平台总交易额 | - |
| 收入 | 平台服务费收入 | - |
| 毛利率 | （收入-成本）/ 收入 | >60% |
| CAC | 单个用户获取成本 | <50元 |
| LTV | 单个用户生命周期价值 | >200元 |

## 四、详细设计

### 4.1 顶部卡片区（4-6个核心指标）

**设计规范：**
- 卡片大小：1/4 宽度
- 布局：4列网格
- 颜色：白底，数字突出，趋势箭头

**示例代码：**
```
┌─────────────────┐
│  📌 累计领养数    │
│     1,256       │
│      只 ⬆️ 15%   │
└─────────────────┘
```

### 4.2 趋势图区

**图表类型：**
| 指标 | 推荐图表 |
|-----|---------|
| 领养量趋势 | 折线图 |
| 用户增长 | 折线图 + 面积图 |
| 转化漏斗 | 漏斗图 |
| 宠物分布 | 饼图/环形图 |
| 收入趋势 | 柱状图 |

**设计规范：**
- 高度：250-300px
- 颜色：品牌色（Rose-500）+ 辅助色
- 交互：悬停显示具体数值
- 筛选：支持日/周/月切换

### 4.3 转化漏斗

**漏斗阶段：**
1. **浏览** - 用户查看宠物详情
2. **咨询** - 用户发起咨询消息
3. **申请** - 用户提交领养申请
4. **签约** - 双方签署领养协议
5. **完成** - 领养交接完成

**计算方式：**
```
转化率 = (当前阶段人数 / 上一阶段人数) × 100%
整体转化率 = (完成人数 / 浏览人数) × 100%
```

### 4.4 用户分析

| 维度 | 指标 |
|-----|------|
| 用户规模 | DAU / MAU / 新增用户 |
| 留存分析 | 次日留存 / 7日留存 / 30日留存 |
| 用户分布 | 城市分布 / 设备分布 / 渠道来源 |
| 用户行为 | 使用时长 / 访问页面数 / 核心功能使用率 |

### 4.5 财务分析

| 维度 | 指标 |
|-----|------|
| 收入 | GMV / 服务费收入 / 会员收入 |
| 成本 | 运营成本 / 获客成本 |
| 利润 | 毛利 / 净利润 |
| 单位经济 | CAC / LTV / 毛利率 |

## 五、技术实现

### 5.1 前端方案

| 方案 | 优点 | 缺点 |
|-----|------|------|
| **React + Recharts** | 生态丰富，灵活度高 | 需要自行实现 |
| **ECharts** | 功能强大，图表类型多 | 体积较大 |
| **DataV（阿里）** | 开箱即用，视觉效果好 | 仅限阿里云 |

### 5.2 数据接口

```
GET /api/dashboard/summary      // 核心指标汇总
GET /api/dashboard/trend        // 趋势数据
GET /api/dashboard/funnel       // 转化漏斗
GET /api/dashboard/users        // 用户数据
GET /api/dashboard/finance      // 财务数据
```

### 5.3 数据更新频率

| 指标类型 | 更新频率 | 说明 |
|---------|---------|------|
| 实时指标 | 实时 | 当前在线、实时订单 |
| 核心指标 | 小时级 | 领养量、DAU 等 |
| 统计指标 | 每日凌晨 | 转化率、留存等 |
| 财务指标 | 每日/每周 | GMV、收入等 |

## 七、投资人展示要点

### 7.1 展示顺序

1. **北极星指标**（领养转化率）
2. **增长趋势**（环比增长）
3. **用户活跃度**（DAU/MAU）
4. **财务健康度**（GMV/毛利率）
5. **运营能力**（留存/满意度）

### 7.2 讲故事的方式

**数据 + 洞察：**
```
❌ "我们的转化率是 18.5%"
✅ "通过优化匹配算法，我们的转化率从 10% 提升到 18.5%
    这意味着每 10 个申请就能多完成 0.85 单领养"
```

**趋势 + 预期：**
```
❌ "用户月活 1234"
✅ "月活从 800 增长到 1234，环比增长 54%
    按这个趋势，下个月将突破 2000"
```
