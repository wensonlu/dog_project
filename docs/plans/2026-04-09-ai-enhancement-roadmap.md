# 汪星球AI增强蓝图 - 三Tab用户参与度提升方案

**设计日期**: 2026-04-09  
**设计目标**: 通过AI增强提升首页/百科/故事三个核心tab的用户参与度

---

## 一、项目背景

### 当前痛点
- **首页**：用户浏览宠物时缺乏引导，匹配效率低，容易流失
- **百科**：知识查找效率低，用户难以快速获得答案
- **故事**：社区互动不足，评论率低，内容供给有限

### 设计原则
- **用户参与度优先**：每个功能都直接提升互动指标
- **组合拳策略**：主动推荐 + 被动响应，快速查询 + 深度学习
- **分阶段实施**：快速验证 → 核心功能 → 增值体验

## 三、阶段1：快速见效（1-2周）

### 3.1 故事tab - AI互动评论生成

#### 产品定位
为领养故事自动生成温暖评论，降低用户互动门槛，活跃社区氛围。

#### 核心功能
1. **智能评论生成**
   - 用户查看故事时，显示"AI推荐评论"按钮
   - 点击后AI分析故事内容，生成3-5条温暖评论选项
   - 用户选择喜欢的评论，一键发布

2. **评论风格多样**
   - 温暖祝福型："太暖心了！希望你们一直幸福下去~"
   - 经验分享型："我家金毛也这样！确实是很好的陪伴犬"
   - 提问互动型："好想知道养金毛一个月要花多少钱？"

3. **情感共鸣引擎**
   - 分析故事关键词：领养、成长、治愈、陪伴
   - 识别情感基调：温馨、感人、励志
   - 生成匹配的评论风格

#### 技术实现
**后端API：**
```javascript
POST /api/stories/:id/generate-comments

请求体：
{
  "storyId": 123,
  "contentType": "adoption_story"
}

响应体：
{
  "success": true,
  "data": {
    "comments": [
      "太暖心了！希望你们一直幸福下去~",
      "我家金毛也这样！确实是很好的陪伴犬",
      "好想知道养金毛一个月要花多少钱？"
    ]
  }
}
```

**AI Prompt设计：**
```
你是一个温暖的宠物社区评论助手。

故事内容：
${storyContent}

请生成3-5条评论，要求：
1. 温暖、真诚、有代入感
2. 鼓励互动和分享
3. 不同的评论风格（祝福、经验分享、提问）
4. 10-30字简短易读

以JSON格式返回：
{
  "comments": ["评论1", "评论2", "评论3"]
}
```

#### 用户流程
```
用户查看故事详情
     ↓
看到评论区冷清（0评论）
     ↓
显示"AI帮我评论"按钮
     ↓
点击按钮，AI生成3条评论选项
     ↓
用户选择喜欢的评论
     ↓
一键发布，评论区活跃起来
```

#### 成功指标
- 评论率提升：150%
- 故事互动率提升：80%
- 用户停留时间提升：40%

### 3.2 首页 - AI实时聊天助手

#### 产品定位
悬浮在首页的AI助手，随时回答用户关于宠物领养的问题，降低选择困难。

#### 核心功能
1. **悬浮助手入口**
   - 右下角显示可爱的狗狗头像按钮
   - 点击展开聊天对话框
   - 可拖动、可最小化

2. **智能对话能力**
   - 品种推荐："我想养一只不拆家的狗" → AI推荐金毛、拉布拉多
   - 养护咨询："第一次养狗需要准备什么？" → AI列出清单
   - 匹配建议："我住公寓，适合养什么狗？" → AI推荐小型犬

3. **上下文记忆**
   - 记住用户之前提过的偏好（喜欢大型犬、有院子）
   - 在推荐宠物时结合历史对话

#### 技术实现
**后端API：**
```javascript
POST /api/ai/chat

请求体：
{
  "userId": "uuid",
  "message": "我想养一只不拆家的狗",
  "context": {
    "previousMessages": [...],
    "userPreferences": {...}
  }
}

响应体：
{
  "success": true,
  "data": {
    "reply": "金毛和拉布拉多都是很好的选择！它们性格温顺，不容易拆家。我给您推荐几只？",
    "suggestedPets": [dogId1, dogId2]
  }
}
```

**AI Prompt设计：**
```
你是汪星球的AI宠物顾问，帮助用户找到合适的宠物。

用户消息：${message}
历史对话：${context}
当前浏览的宠物：${currentPets}

回复要求：
1. 热情、专业、有耐心
2. 给出具体建议，不要模棱两可
3. 可以推荐当前平台的宠物
4. 控制在100字以内，简洁有力
```

#### 用户流程
```
用户浏览首页宠物
     ↓
看到右下角AI助手图标
     ↓
点击打开聊天窗口
     ↓
输入："我想养一只不掉毛的狗"
     ↓
AI回答："贵宾犬和比熊都不掉毛！给您推荐几只"
     ↓
AI在对话中展示推荐的宠物卡片
     ↓
用户点击卡片跳转到宠物详情
```

#### 成功指标
- AI助手使用率：40%
- 对话转宠物查看率：50%
- 新用户转化率提升：40%

## 四、阶段2：核心功能（2-3周）

### 4.1 首页 - AI宠物匹配顾问

#### 产品定位
基于用户滑动行为，实时分析偏好，主动推荐匹配度高的宠物。

#### 核心功能
1. **行为数据收集**
   - 记录滑动行为：喜欢、跳过、收藏、查看详情
   - 提取偏好特征：品种偏好、年龄偏好、体型偏好
   - 生成用户画像："喜欢中型犬、偏向年轻狗狗"

2. **实时推荐引擎**
   - 每滑动10张卡片，AI分析一次偏好
   - 弹出推荐卡片："为您推荐3只合适的宠物"
   - 显示匹配理由："这只金毛符合您对大型犬的偏好"

3. **推荐理由生成**
   - "您喜欢活泼的狗狗，这只柴犬精力充沛，很适合您"
   - "您偏向年轻狗狗，这只1岁的边牧正值黄金年龄"
   - "您关注了3只金毛，这只纯种金毛值得一看"

#### 技术实现
**推荐算法：**
```javascript
// 偏好提取
function extractUserPreferences(swipeHistory) {
  return {
    breedPreference: analyzeBreedPreference(swipeHistory),
    agePreference: analyzeAgePreference(swipeHistory),
    sizePreference: analyzeSizePreference(swipeHistory),
    activityLevel: analyzeActivityLevel(swipeHistory)
  };
}

// 匹配度计算
function calculateMatchScore(dog, preferences) {
  const breedScore = calculateBreedScore(dog.breed, preferences.breedPreference);
  const ageScore = calculateAgeScore(dog.age, preferences.agePreference);
  const sizeScore = calculateSizeScore(dog.size, preferences.sizePreference);
  
  return (breedScore + ageScore + sizeScore) / 3;
}
```

**AI推荐理由生成：**
```javascript
POST /api/ai/recommendation-reason

请求体：
{
  "dog": {...},
  "userPreferences": {...}
}

响应体：
{
  "reason": "这只金毛符合您对大型犬的偏好，且性格温顺，适合第一次养狗的您"
}
```

#### 成功指标
- 推荐接受率：60%
- 匹配成功率提升：30%
- 用户平均滑动次数提升：50%

### 4.2 百科 - AI问答助手

#### 产品定位
集成RAG技术，让用户直接提问获得答案，无需翻阅多篇文章。

#### 核心功能
1. **智能问答界面**
   - 百科首页顶部显示搜索框
   - 用户输入问题："金毛掉毛严重怎么办？"
   - AI即时回答，并推荐相关文章

2. **RAG知识检索**
   - 从百科文章库中检索相关段落
   - AI基于检索内容生成准确答案
   - 引用来源文章，可点击查看原文

3. **多轮对话**
   - 用户追问："除了梳毛还有其他办法吗？"
   - AI理解上下文，继续回答
   - 生成相关推荐："您可能还想了解：金毛毛发护理指南"

#### 技术实现
**RAG架构：**
```
用户问题
    ↓
Embedding向量化
    ↓
在Supabase中检索相关文章片段
    ↓
将片段作为Context传递给AI
    ↓
AI生成答案并引用来源
```

**API设计：**
```javascript
POST /api/wiki/ai-qa

请求体：
{
  "question": "金毛掉毛严重怎么办？",
  "conversationHistory": [...]
}

响应体：
{
  "success": true,
  "data": {
    "answer": "金毛掉毛严重可以通过以下方法缓解：1. 每天梳毛2-3次...",
    "sourceArticles": [
      {title: "金毛毛发护理指南", url: "/wiki/article/123"}
    ],
    "relatedQuestions": [
      "金毛什么时候换毛？",
      "金毛用什么梳子最好？"
    ]
  }
}
```

#### 成功指标
- 问题解决率：85%
- 查询效率提升：60%
- 用户停留时间提升：3倍

## 五、阶段3：增值体验（3-4周）

### 5.1 百科 - AI知识图谱可视化

#### 产品定位
将百科知识以图谱形式展示，帮助用户理解知识关联。

#### 核心功能
1. **知识图谱构建**
   - 从文章中提取实体：疾病、症状、治疗方法、品种特性
   - 建立实体关系：驱虫 → 预防 → 心丝虫病
   - 生成可视化图谱

2. **交互式探索**
   - 点击节点展开相关知识
   - 拖拽、缩放查看图谱全貌
   - 高亮显示学习路径

3. **AI学习路径推荐**
   - "新手养狗必学知识路径"：选犬 → 驱虫 → 疫苗 → 绝育
   - 根据用户浏览历史推荐个性化学习路径

#### 技术实现
**知识图谱数据结构：**
```javascript
{
  "nodes": [
    {"id": "驱虫", "type": "treatment"},
    {"id": "心丝虫病", "type": "disease"},
    {"id": "症状", "type": "symptom"}
  ],
  "edges": [
    {"source": "驱虫", "target": "心丝虫病", "relation": "预防"},
    {"source": "心丝虫病", "target": "症状", "relation": "has"}
  ]
}
```

**可视化库：** D3.js 或 ECharts Graph

#### 成功指标
- 知识深度学习率：40%
- 平均学习时长：10分钟
- 知识关联理解度提升：80%
